# 🎓 Enregistrement des Étudiants – Projet VBA Excel

## 📌 Description

Ce projet a été réalisé dans le cadre d’un travail académique collaboratif.
Il s'agit d'un système simple d'enregistrement des étudiants pour le paiement de leur enrôlement.
L’interface est construite avec un **formulaire VBA** dans Excel, reliée à une feuille servant de **base de données**.

---

## 🛠️ Fonctionnalités

- ✅ **Ajout d’un étudiant** via un formulaire
- 🔍 **Recherche** par matricule
- ✏️ **Modification** des informations d’un étudiant existant
- 🗑️ **Suppression** d’un étudiant de la base
- 🔐 **Validation** des champs obligatoires (message d’erreur si vide ou invalide)
- 🔁 **Contrôle d’unicité** des matricules (aucun doublon autorisé)

---

## 📂 Organisation des fichiers

- `Formulaire` : interface d’enregistrement ( Un peu comme un UserForm)
- `Données` : feuille servant de base de données interne
- `Macros` : code VBA assurant le fonctionnement des boutons

---

## 🧠 Logique technique

- 📌 Toutes les actions passent par des **macros VBA**
- 📅 Les dates sont vérifiées au moment de l’enregistrement
- 🧾 Un message d'erreur apparaît si des champs sont laissés vides ou si les règles ne sont pas respectées

---

## 🚀 Perspectives d’amélioration
